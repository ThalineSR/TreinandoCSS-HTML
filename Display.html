Pripriedade display: inline-block;

Trata o elemento como um elemento em linha, mas com a capacidade de definir largura, altura, margens e preenchimento como um elemento em bloco. Isso significa que ele ocupa apenas o espaço necessário na linha, mas você pode definir dimensões e espaçamento para ele como faria com um elemento de bloco normal. Isso é útil quando você deseja elementos que fluam na mesma linha, mas com mais controle sobre seu layout em comparação com elementos puramente em linha.

Pripriedade display: none;

Quando você define display: none; em CSS, está basicamente dizendo ao navegador para não exibir o elemento na página. Ele não ocupará espaço no layout e não será renderizado visualmente na página. Isso pode ser útil para ocultar temporariamente elementos que não são necessários para a visualização atual ou para ocultar elementos em certas condições usando JavaScript.

display: none vs visibility: hidden

Quando você usa visibility: hidden; em CSS, você está tornando o elemento invisível, mas ele ainda ocupa espaço na página, ou seja, o espaço que o elemento ocuparia ainda está reservado, mesmo que não seja visível. Isso pode ser útil quando você quer esconder um elemento sem alterar o layout da página. Por exemplo, você pode querer esconder um submenu que será revelado quando o usuário interagir com um elemento específico.

Alinhamento de elementos inline

text-aling

A propriedade CSS text-align é usada para alinhar o texto dentro de um elemento. Ela define o alinhamento horizontal do conteúdo de texto dentro do elemento, como um _div_ ou um _p_. Você pode usar essa propriedade para alinhar o texto à esquerda, à direita, no centro ou justificado dentro do elemento.

      text-align: left; alinha o texto à esquerda.
      text-align: right; alinha o texto à direita.
      text-align: center; centraliza o texto.
      text-align: justify; justifica o texto, preenchendo o espaço disponível no elemento, exceto a última linha, que é alinhada à esquerda.

vertical-align

A propriedade vertical-align em CSS controla a forma como um elemento inline ou um elemento de tabela é alinhado verticalmente em relação ao elemento pai ou a outros elementos em torno dele. Essa propriedade é mais comumente usada em elementos de texto inline ou em células de tabela.

      vertical-align: top; alinha o elemento ao topo do elemento pai.
      vertical-align: middle; o alinha verticalmente ao meio do elemento pai.
      vertical-align: bottom; o alinha verticalmente à base do elemento pai.
      vertical-align: baseline; o alinha verticalmente à linha de base do elemento pai.

    Esse é o valor padrão. Elementos com position: static; são posicionados de acordo com o fluxo normal do documento. As propriedades top, right, bottom e left não têm efeito sobre elementos com position: static;.

Propriedade z-index

A propriedade z-index em CSS é usada para controlar a ordem de empilhamento de elementos posicionados que se sobrepõem. Ela só tem efeito em elementos que tenham um valor de position diferente de static (por exemplo, relative, absolute, fixed, ou sticky). Quando dois ou mais elementos se sobrepõem em uma página da web, a propriedade z-index determina qual elemento deve aparecer sobre o outro. Quanto maior o valor de z-index, mais alto o elemento é empilhado na ordem de empilhamento e, portanto, mais próximo do topo da pilha.
