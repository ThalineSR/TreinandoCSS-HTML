Propriedade background-color

A propriedade CSS background-color é utilizada para definir a cor de fundo de um elemento. Essa propriedade pode ser aplicada a praticamente qualquer elemento HTML para alterar a aparência visual do conteúdo e dar destaque ou contraste ao design da página. A background-color pode ser especificada utilizando diferentes formatos de valores, como nomes de cores, valores hexadecimais, RGB, RGBA, HSL, e HSLA.

Exemplos de Uso:

  • Nome da Cor:
    background-color: red;
  • Valor Hexadecimal:
    background-color:
    #ff0000;
  • RGB:
    background-color:
    rgb(255, 0, 0);
  • RGBA (com opacidade):
    background-color:
    rgba(255, 0, 0, 0.5);
  • HSL:
    background-color:
    hsl(0, 100%, 50%);
  • HSLA (com opacidade):
    background-color:
    hsla(0, 100%, 50%, 0.5);

background-image

A propriedade background-image no CSS é usada para definir uma ou mais imagens de fundo para um elemento. Essas imagens podem ser arquivos de imagem, gradientes ou outros tipos de imagens suportados pelo CSS. A utilização adequada de background-image pode adicionar um apelo visual significativo às suas páginas web.

Sintaxe Básica background-image: url('imagem.jpg');

Valore:

  • Gradientes:
      Radial Gradient:background-image: radial-gradient(circle, red, blue);
      Linear Gradient:background-image: linear-gradient(to right, red, blue);
  • URL de Imagem: background-image: url('imagem.jpg');
  • Múltiplas Imagens: Permite definir múltiplas imagens de fundo, que serão empilhadas umas sobre as outras, a primeira definida será a mais próxima do usuário. background-image: url('imagem1.jpg'), linear-gradient(to right, red, blue); background-blend-mode:multiply;

Propriedade back-size

Ela permite que você ajuste a imagem para cobrir o elemento de diferentes maneiras, garantindo que o design da página seja responsivo e visualmente agradável.

Valores

  • Background-size: Auto;

    Mantém o tamanho original da imagem.

  • Background-size: cover;

    Redimensiona a imagem para cobrir completamente o elemento, mantendo a proporção original da imagem. A imagem pode ser cortada para preencher o elemento.

  • Background-size: contain;

    Redimensiona a imagem para caber dentro do elemento, mantendo a proporção original da imagem. A imagem será dimensionada para garantir que toda ela seja visível.

  • Background-size: valor;

    Define o tamanho da imagem de fundo em unidades como pixels, porcentagem, etc.

  • Background-size: x y; (dois valores)

    Define o tamanho da imagem de fundo em unidades como pixels, porcentagem, etc.

propriedade background-origin

A propriedade CSS background-origin especifica a caixa de referência usada para calcular a posição inicial de uma imagem de fundo (background) em um elemento. Ela determina onde a imagem de fundo começará a ser exibida em relação à caixa do elemento.

Os três valores possíveis para background-origin são:

  • padding-box: A imagem de fundo começa a ser exibida a partir das bordas internas da área de preenchimento (padding) do elemento. Isso significa que a imagem de fundo será cortada para se ajustar à área de preenchimento do elemento.

    background-origin: padding-box;

  • border-box: A imagem de fundo começa a ser exibida a partir das bordas da borda (border) do elemento. Isso significa que a imagem de fundo será cortada para se ajustar à área de preenchimento e à borda do elemento.

    background-origin: border-box;

  • content-box: A imagem de fundo começa a ser exibida a partir das bordas externas do conteúdo (content) do elemento. Isso significa que a imagem de fundo será exibida sem ser cortada e será posicionada em relação ao conteúdo do elemento.background-origin: content-box;

Propriedade background-clip

A propriedade background-clip no CSS é utilizada para definir até onde a imagem ou cor de fundo de um elemento deve se estender dentro do modelo de caixa (box model) do elemento. Isso permite um controle detalhado sobre a área de pintura do fundo, que pode incluir a borda, o preenchimento (padding), ou apenas o conteúdo do elemento.

Sintaxe: background-clip: valor;

Valores:

  • O fundo se estende até a borda interna do padding, excluindo a borda. background-clip: padding-box;

    Trilha CSS

  • O fundo se estende até a borda externa do elemento, incluindo a borda. background-clip: border-box;

    Trilha CSS

  • O fundo se estende apenas até a borda do conteúdo, excluindo o padding e a borda. background-clip: content-box;

    Trilha CSS

  • O fundo se aplica ao texto dentro do elemento. Este valor é menos comum e só é suportado em certos navegadores. background-clip; text
