Alinhamento de Texto (text-aling)

Essa propriedade é responsavel por alinhar nosso texto no nosso elemento.

text-align: left;

left: Alinha o texto à esquerda.

text-align: right;

right: Alinha o texto à direita.

text-align: center;

center: Alinha o texto no centro.

text-align: justify;

justify: Distribui o texto uniformemente entre as margens esquerda e direita, criando espaços adicionais entre as palavras, exceto a última linha.

text-align: initial;

initial: Define a propriedade para o seu valor inicial.

text-align: inherit;

inherit: Herda o valor da propriedade de seu pai.

Decoração de texto (text-decoration)

Adiciona ou remove linhas do nosso texto

text-decoration: none

Remove qualquer decoração de texto.

text-decoration: underline

Adiciona um sublinhado ao texto.

text-decoration: overline

Adiciona uma linha sobre o texto.

text-decoration: line-through

Adiciona uma linha através do texto.

text-decoration: underline overline

Adiciona um sublinhado e uma linha sobre o texto.

text-decoration: underline line-through

Adiciona um sublinhado e uma linha através do texto.

text-decoration: blink

Faz com que o texto pisque ou alterne entre aparecer e desaparecer. Este valor é menos comum e pode não ser suportado em todos os navegadores.

É uma propriedade abreviada de:

  • text-decoration-line

    Ela permite especificar diretamente qual tipo de decoração de linha deseja-se aplicar.

        text-decoration-line: none;
        text-decoration-line: underline;
        text-decoration-line: overline;
        text-decoration-line: line-through;
        text-decoration-line: underline overline;
        text-decoration-line: underline line-through;

  • text-decoration-style

    É usada para definir o estilo da linha de decoração de texto.

    text-decoration-style: solid;

    Define uma linha de decoração sólida.

    text-decoration-style: double;

    Define uma linha de decoração dupla.

    text-decoration-style: dotted;

    Define uma linha de decoração pontilhada.

    text-decoration-style: dashed;

    Define uma linha de decoração tracejada

    text-decoration-style: wavy;

    Define uma linha de decoração ondulada.


  • text-decoration-color

    É usada para definir a cor da linha de decoração de texto.

    text-decoration-line: underline; /* Define o tipo de decoração */
    text-decoration-color: red; /* Define a cor da decoração */

    Exemplo


  • text-decoration-thickness

    Define a espessura da linha de decoração de texto.

    text-decoration-line: underline; /* Define o tipo de decoração */
    text-decoration-thickness: 5px; /* Define a espessura da decoração */

    Exemplo


Em resumo podemos usar da seguinte forma

text-decoration: underline solid red 1px;

Propriedade text-indent

Define o recuo da primeira linha do nosso texto

text-indent: 20px;

Neste exemplo, a primeira linha de texto dentro do elemento terá um recuo de 20 pixels em relação à margem esquerda padrão. Aceita tambem numeros negativos.

Propriedade: letter-spacing

Adiciona um espaço entre cada CARACTERE do seu texto. Como o nome diz espaçamento entre letras. Aceita valores negativos e positivos.

Propriedade: word-spacing

Adiciona um espaço entre cada PALAVRA do seu texto. Como o nome diz espaçamento entre palavras. Aceita valores negativos e positivos.
