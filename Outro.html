Propriedade opacity

Define o nivel de opacidade (transparência) de um elemento.

Exemplo:

opacity: .3;

opacity: .10;

opacity: .0;

Propriedade overflow

Controla como o conteudo deve ser exibido caso ultrapasse os limites do container

Exemplo:

  • visible: Este é o valor padrão. Ele faz com que o conteúdo que excede as dimensões do elemento seja exibido fora das bordas do elemento, sem ser recortado ou oculto.
  • hidden: Conteúdo que excede as dimensões do elemento será simplesmente ocultado. Isso significa que ele não será exibido e não estará acessível visualmente para o usuário.
  • scroll: Uma barra de rolagem será adicionada ao elemento, permitindo que o usuário role para ver o conteúdo oculto.
  • auto: Similar ao scroll, mas uma barra de rolagem só é adicionada quando o conteúdo excede as dimensões do elemento. Se o conteúdo não ultrapassar as dimensões, nenhuma barra de rolagem será adicionada.
