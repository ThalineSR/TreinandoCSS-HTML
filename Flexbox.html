Os Eixos do flexbox

No Flexbox, trabalhamos com dois eixos principais: o eixo principal (main axis) e o eixo transversal (cross axis). Esses eixos determinam como os itens flexíveis são dispostos dentro do contêiner flexível. Aqui está uma visão geral dos eixos do Flexbox:

Eixo Principal (Main Axis):

  • O eixo principal é determinado pela propriedade flex-direction.
  • Se flex-direction for definido como row ou row-reverse, o eixo principal será horizontal, indo da esquerda para a direita.
  • Se flex-direction for definido como column ou column-reverse, o eixo principal será vertical, indo de cima para baixo.

Eixo Transversal (Cross Axis):

  • O eixo transversal é perpendicular ao eixo principal.
  • Se flex-direction for row ou row-reverse, o eixo transversal será vertical, indo de cima para baixo.
  • Se flex-direction for column ou column-reverse, o eixo transversal será horizontal, indo da esquerda para a direita.

Esses eixos são fundamentais para entender como os itens flexíveis são posicionados e alinhados dentro do contêiner flexível. As propriedades justify-content e align-items controlam o alinhamento dos itens ao longo do eixo principal e do eixo transversal, respectivamente. Entender e manipular esses eixos é essencial para criar layouts flexíveis e responsivos com Flexbox.

Juntando as propriedades Flex-direction e flex-wrap

flex-flow

A propriedade flex-flow é uma abreviação que combina as propriedades flex-direction e flex-wrap em uma única declaração. Isso permite definir a direção dos itens flexíveis e se eles devem ou não serem dispostos em uma única linha ou várias linhas em um contêiner flexível em uma única linha de código.

A sintaxe da propriedade flex-flow é a seguinte:

flex-flow: < flex-direction > < flex-wrap >;
    Onde:
  • < flex-direction > é a direção principal dos itens flexíveis (como row, row-reverse, column ou column-reverse).
  • < flex-wrap > é a configuração de embrulho dos itens flexíveis (como nowrap, wrap ou wrap-reverse).

Exemplo:

display: flex;

Propriedade order

No contexto do Flexbox, a propriedade order é usada para especificar a ordem em que os itens flexíveis devem ser exibidos no contêiner flexível. Por padrão, os itens flexíveis são exibidos na ordem em que aparecem no HTML, mas a propriedade order permite reorganizá-los. Ela aceita valores numéricos, onde itens com um valor de order menor são exibidos primeiro e itens com um valor maior são exibidos depois. Se dois itens têm o mesmo valor de order, eles são exibidos na ordem em que aparecem no HTML. No contexto do Grid, a propriedade order não é aplicável, pois a ordem dos itens em uma grade é determinada pelo posicionamento deles na estrutura de grade, não por uma propriedade específica de ordem.

Propriedade flex-grow

Ela determina a capacidade de um item flexível expandir-se para preencher o espaço disponível dentro de um contêiner flexível. Essa propriedade aceita um valor numérico que representa a proporção em que o item deve crescer em relação aos outros itens flexíveis dentro do mesmo contêiner. Quanto maior o valor de flex-grow para um item em comparação com outros itens, mais espaço ele ocupará em relação a eles quando houver espaço extra disponível. Se todos os itens em um contêiner flexível tiverem o mesmo valor de flex-grow, eles crescerão igualmente para preencher o espaço disponível. Se um item tiver um valor de flex-grow maior que os outros, ele ocupará uma parcela maior do espaço extra.

Propriedade flex-basis

A propriedade flex-basis é usada em layouts flexíveis (flexbox) em CSS para definir o tamanho inicial de um item flexível antes que o espaço restante seja distribuído. Ela especifica o tamanho base de um item flexível no contêiner flexível, antes que o espaço adicional seja distribuído de acordo com as proporções definidas pelas propriedades flex-grow e flex-shrink. O valor padrão para flex-basis é auto, o que significa que o tamanho inicial do item será o seu tamanho intrínseco (se houver) ou o tamanho determinado pelo valor width ou height, dependendo da direção principal do flex container. Por exemplo, se flex-direction for row, flex-basis controlará a largura inicial do item. Se flex-direction for column, flex-basis controlará a altura inicial do item. Você pode definir o valor de flex-basis como um valor específico de largura, altura ou porcentagem, ou usar palavras-chave como auto ou content.

Propriedade shorthand-flex

A propriedade shorthand flex é uma maneira conveniente de definir as três propriedades flexíveis: flex-grow, flex-shrink e flex-basis, tudo em uma única linha.

A sintaxe geral é:

flex: [flex-grow] [flex-shrink] [flex-basis];

Por exemplo, se você definir flex: 1 1 auto; significa que o item flexível pode crescer (flex-grow: 1), pode encolher (flex-shrink: 1) e o tamanho inicial será determinado automaticamente pelo navegador (flex-basis: auto;). Se você fornecer apenas um valor numérico, ele será interpretado como o valor de flex-grow. Se fornecer dois valores numéricos, o primeiro será flex-grow e o segundo será flex-shrink, e o terceiro valor será flex-basis. Se fornecer um valor seguido pela palavra-chave auto, isso definirá flex-grow e flex-basis, com flex-shrink sendo 1. Se você fornecer apenas um valor seguido por auto, o valor será interpretado como flex-grow, e o flex-basis será definido como auto. É uma ótima maneira de simplificar o código CSS quando você precisa definir as propriedades flexíveis de uma só vez.

Exemplo:

flex: 1 1 auto;

Propriedade aling-self

A propriedade align-self é uma propriedade CSS usada em layouts flexíveis (flexbox) para controlar o alinhamento de um único item flexível em relação ao eixo transversal do contêiner flexível. Enquanto a propriedade align-items é usada para alinhar todos os itens flexíveis dentro do contêiner, a propriedade align-self permite ajustar individualmente o alinhamento de um item específico.
