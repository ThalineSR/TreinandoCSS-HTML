Propriedade: Object-fit

Essa propriedade vai dizer se a imagem deve ser distorcida para caber no objeto ou se ela deve se manter no seu tamanho, respeitando ou não o objeto, podendo assim alterar a qualidade da imagem.

Valores:

  • object-fit: fill; (Padrao)
  • object-fit: contain;
  • object-fit: cover;
  • object-fit: none;
  • object-fit:scale-down ;

object-position

A propriedade CSS object-position é usada em conjunto com object-fit para definir a posição de uma imagem ou vídeo dentro do contêiner, quando o tamanho do conteúdo é diferente do tamanho do contêiner.

Sintaxe: object-position: x y;

x: Posição horizontal (pode ser um valor de comprimento, porcentagem, ou palavra-chave). y: Posição vertical (pode ser um valor de comprimento, porcentagem, ou palavra-chave). Valores Comuns Palavras-chave predefinidas: left, center, right (horizontal) top, center, bottom (vertical) Porcentagem: 0% representa o início (esquerda ou topo). 100% representa o fim (direita ou fundo). 50% representa o meio. Comprimento: Pode ser especificado em unidades como px, em, rem, etc.

Propriedade background-color

A propriedade CSS background-color é utilizada para definir a cor de fundo de um elemento. Essa propriedade pode ser aplicada a praticamente qualquer elemento HTML para alterar a aparência visual do conteúdo e dar destaque ou contraste ao design da página. A background-color pode ser especificada utilizando diferentes formatos de valores, como nomes de cores, valores hexadecimais, RGB, RGBA, HSL, e HSLA.

Exemplos de Uso:

  • Nome da Cor:
    background-color: red;
  • Valor Hexadecimal:
    background-color: #ff0000;
  • RGB:
    background-color: rgb(255, 0, 0);
  • RGBA (com opacidade):
    background-color: rgba(255, 0, 0, 0.5);
  • HSL:
    background-color: hsl(0, 100%, 50%);
  • HSLA (com opacidade):
    background-color: hsla(0, 100%, 50%, 0.5);

background-image

A propriedade background-image no CSS é usada para definir uma ou mais imagens de fundo para um elemento. Essas imagens podem ser arquivos de imagem, gradientes ou outros tipos de imagens suportados pelo CSS. A utilização adequada de background-image pode adicionar um apelo visual significativo às suas páginas web.

Sintaxe Básica: background-image: url('imagem.jpg');

Valores:


  • Radial Gradient:
    background-image: radial-gradient(circle, red, blue);

  • Linear Gradient:
    background-image: linear-gradient(to right, red, blue);
  • URL de Imagem:
    background-image: url('imagem.jpg');

  • Múltiplas Imagens:
    background-image: url('imagem1.jpg'), linear-gradient(to right, red, blue);
    background-blend-mode:multiply;

background-repeat

É usada para definir como uma imagem de fundo deve se repetir dentro de um elemento. Ela pode especificar se a imagem de fundo deve ser repetida horizontalmente, verticalmente, em ambas as direções, ou não ser repetida.

Valores:

  • background-repeat: repeat (valor padrão);
  • background-repeat: repeat-x ;
  • background-repeat: repeat-y ;
  • background-repeat: space;
  • background-repeat: round;
  • background-repeat: no-repeat;
  • background-repeat: dois valores;

background-position

Esta propriedade permite especificar a posição inicial da imagem de fundo e pode ser definida utilizando valores de comprimento, porcentagem, ou palavras-chave predefinidas.

Valores:

  • background-position: bottom; (valor padrão);
  • background-position: top;
  • background-position: left;
  • background-position:right
  • background-position: valor (30px);
  • background-position: center;
  • background-position: dois valores;
  • background-position: valores de cantos relativos; y, x

Propriedade back-size

Ela permite que você ajuste a imagem para cobrir o elemento de diferentes maneiras, garantindo que o design da página seja responsivo e visualmente agradável.

Valores:

  • Background-size: Auto;

    Mantém o tamanho original da imagem.

  • Background-size: cover;

    Redimensiona a imagem para cobrir completamente o elemento, mantendo a proporção original da imagem. A imagem pode ser cortada para preencher o elemento.

  • Background-size: contain;

    Redimensiona a imagem para caber dentro do elemento, mantendo a proporção original da imagem. A imagem será dimensionada para garantir que toda ela seja visível.

  • Background-size: valor;

    Define o tamanho da imagem de fundo em unidades como pixels, porcentagem, etc.

  • Background-size: x y; (dois valores)

    Define o tamanho da imagem de fundo em unidades como pixels, porcentagem, etc.

propriedade background-origin

A propriedade CSS background-origin especifica a caixa de referência usada para calcular a posição inicial de uma imagem de fundo (background) em um elemento. Ela determina onde a imagem de fundo começará a ser exibida em relação à caixa do elemento.

Os três valores possíveis para background-origin são:

  • padding-box: A imagem de fundo começa a ser exibida a partir das bordas internas da área de preenchimento (padding) do elemento. Isso significa que a imagem de fundo será cortada para se ajustar à área de preenchimento do elemento.

    background-origin: padding-box;

  • border-box: A imagem de fundo começa a ser exibida a partir das bordas da borda (border) do elemento. Isso significa que a imagem de fundo será cortada para se ajustar à área de preenchimento e à borda do elemento.

    background-origin: border-box;

  • content-box: A imagem de fundo começa a ser exibida a partir das bordas externas do conteúdo (content) do elemento. Isso significa que a imagem de fundo será exibida sem ser cortada e será posicionada em relação ao conteúdo do elemento.

    background-origin: content-box;

Propriedade background-clip

A propriedade background-clip no CSS é utilizada para definir até onde a imagem ou cor de fundo de um elemento deve se estender dentro do modelo de caixa (box model) do elemento. Isso permite um controle detalhado sobre a área de pintura do fundo, que pode incluir a borda, o preenchimento (padding), ou apenas o conteúdo do elemento.

Sintaxe: background-clip: valor;

Valores:

  • O fundo se estende até a borda interna do padding, excluindo a borda.

    background-clip: padding-box;
    Trilha CSS

  • O fundo se estende até a borda externa do elemento, incluindo a borda.

    background-clip: border-box;
    Trilha CSS

  • O fundo se estende apenas até a borda do conteúdo, excluindo o padding e a borda.

    background-clip: content-box;
    Trilha CSS

  • O fundo se aplica ao texto dentro do elemento. Este valor é menos comum e só é suportado em certos navegadores.

    background-clip; text
    Trilha CSS

Para resumir uma unidade background em CSS, geralmente seguimos uma ordem lógica para garantir que todas as propriedades relevantes sejam definidas de forma clara e organizada. Aqui está uma ordem comum para resumir uma unidade background:

  1. background-color: Define a cor de fundo.
  2. background-image: Especifica a imagem de fundo, se houver.
  3. background-repeat: Define como a imagem de fundo deve se repetir.
  4. background-position: Define a posição inicial da imagem de fundo.
  5. background-size: Especifica o tamanho da imagem de fundo.
  6. background-attachment: Define se a imagem de fundo deve se mover com o conteúdo ou permanecer fixa enquanto a página é rolada.
  7. background-clip: Especifica a área de recorte para o plano de fundo, se necessário.
  8. background-origin: Define a posição de origem do plano de fundo em relação à caixa do elemento, se necessário.
  9. background-blend-mode: Especifica o modo de mesclagem entre a imagem de fundo e o conteúdo, se necessário.
