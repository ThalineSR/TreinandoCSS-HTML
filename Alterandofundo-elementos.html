Propriedade: Object-fit

Essa propriedade vai dizer se a imagem deve ser distorcida para caber no objeto ou se ela deve se manter no seu tamanho, respeitando ou não o objeto, podendo assim alterar a qualidade da imagem.

Valores:

  • object-fit: fill; (Padrao)
  • object-fit: contain;
  • object-fit: cover;
  • object-fit: none;
  • object-fit:scale-down ;

object-position

A propriedade CSS object-position é usada em conjunto com object-fit para definir a posição de uma imagem ou vídeo dentro do contêiner, quando o tamanho do conteúdo é diferente do tamanho do contêiner.

Sintaxe: object-position: x y;

x: Posição horizontal (pode ser um valor de comprimento, porcentagem, ou palavra-chave). y: Posição vertical (pode ser um valor de comprimento, porcentagem, ou palavra-chave). Valores Comuns Palavras-chave predefinidas: left, center, right (horizontal) top, center, bottom (vertical) Porcentagem: 0% representa o início (esquerda ou topo). 100% representa o fim (direita ou fundo). 50% representa o meio. Comprimento: Pode ser especificado em unidades como px, em, rem, etc.

Propriedade background-color

A propriedade CSS background-color é utilizada para definir a cor de fundo de um elemento. Essa propriedade pode ser aplicada a praticamente qualquer elemento HTML para alterar a aparência visual do conteúdo e dar destaque ou contraste ao design da página. A background-color pode ser especificada utilizando diferentes formatos de valores, como nomes de cores, valores hexadecimais, RGB, RGBA, HSL, e HSLA.

Exemplos de Uso:

  • Nome da Cor:
    background-color: red;
  • Valor Hexadecimal:
    background-color: #ff0000;
  • RGB:
    background-color: rgb(255, 0, 0);
  • RGBA (com opacidade):
    background-color: rgba(255, 0, 0, 0.5);
  • HSL:
    background-color: hsl(0, 100%, 50%);
  • HSLA (com opacidade):
    background-color: hsla(0, 100%, 50%, 0.5);

background-image

A propriedade background-image no CSS é usada para definir uma ou mais imagens de fundo para um elemento. Essas imagens podem ser arquivos de imagem, gradientes ou outros tipos de imagens suportados pelo CSS. A utilização adequada de background-image pode adicionar um apelo visual significativo às suas páginas web.

Sintaxe Básica: background-image: url('imagem.jpg');

Valores:


  • Radial Gradient:
    background-image: radial-gradient(circle, red, blue);

  • Linear Gradient:
    background-image: linear-gradient(to right, red, blue);
  • URL de Imagem:
    background-image: url('imagem.jpg');

  • Múltiplas Imagens:
    background-image: url('imagem1.jpg'), linear-gradient(to right, red, blue);
    background-blend-mode:multiply;

  • background-repeat: repeat-x ;
  • background-repeat: repeat-y ;
  • background-repeat: space;
  • background-repeat: round;
  • background-repeat: no-repeat;
  • background-repeat: dois valores;

propriedade background-origin

A propriedade CSS background-origin especifica a caixa de referência usada para calcular a posição inicial de uma imagem de fundo (background) em um elemento. Ela determina onde a imagem de fundo começará a ser exibida em relação à caixa do elemento.

Os três valores possíveis para background-origin são:

  • padding-box: A imagem de fundo começa a ser exibida a partir das bordas internas da área de preenchimento (padding) do elemento. Isso significa que a imagem de fundo será cortada para se ajustar à área de preenchimento do elemento.

    background-origin: padding-box;

  • border-box: A imagem de fundo começa a ser exibida a partir das bordas da borda (border) do elemento. Isso significa que a imagem de fundo será cortada para se ajustar à área de preenchimento e à borda do elemento.

    background-origin: border-box;

  • content-box: A imagem de fundo começa a ser exibida a partir das bordas externas do conteúdo (content) do elemento. Isso significa que a imagem de fundo será exibida sem ser cortada e será posicionada em relação ao conteúdo do elemento.

    background-origin: content-box;

Propriedade background-clip

A propriedade background-clip no CSS é utilizada para definir até onde a imagem ou cor de fundo de um elemento deve se estender dentro do modelo de caixa (box model) do elemento. Isso permite um controle detalhado sobre a área de pintura do fundo, que pode incluir a borda, o preenchimento (padding), ou apenas o conteúdo do elemento.

Sintaxe: background-clip: valor;

Valores:

  • O fundo se estende até a borda interna do padding, excluindo a borda.

    background-clip: padding-box;
    Trilha CSS

  • O fundo se estende até a borda externa do elemento, incluindo a borda.

    background-clip: border-box;
    Trilha CSS

  • O fundo se estende apenas até a borda do conteúdo, excluindo o padding e a borda.

    background-clip: content-box;
    Trilha CSS

  • O fundo se aplica ao texto dentro do elemento. Este valor é menos comum e só é suportado em certos navegadores.

    background-clip; text
    Trilha CSS

Para resumir uma unidade background em CSS, geralmente seguimos uma ordem lógica para garantir que todas as propriedades relevantes sejam definidas de forma clara e organizada. Aqui está uma ordem comum para resumir uma unidade background:

  1. background-color: Define a cor de fundo.
  2. background-image: Especifica a imagem de fundo, se houver.
  3. background-repeat: Define como a imagem de fundo deve se repetir.
  4. background-position: Define a posição inicial da imagem de fundo.
  5. background-size: Especifica o tamanho da imagem de fundo.
  6. background-attachment: Define se a imagem de fundo deve se mover com o conteúdo ou permanecer fixa enquanto a página é rolada.
  7. background-clip: Especifica a área de recorte para o plano de fundo, se necessário.
  8. background-origin: Define a posição de origem do plano de fundo em relação à caixa do elemento, se necessário.
  9. background-blend-mode: Especifica o modo de mesclagem entre a imagem de fundo e o conteúdo, se necessário.

Font-family

Em CSS, a propriedade font-family é usada para especificar a família de fontes que será aplicada a um elemento HTML. Uma família de fontes consiste em um conjunto de fontes semelhantes que compartilham um design básico, mas podem variar em estilo, peso e largura.


Quando você define a propriedade font-family, você está instruindo o navegador a procurar por uma fonte específica no sistema do usuário para exibir o texto. Se a fonte especificada não estiver disponível no sistema, o navegador tentará encontrar uma alternativa similar na lista de fontes fornecida.


A propriedade font-family pode conter vários valores de fonte, separados por vírgulas. Isso permite que você forneça uma lista de fontes de fallback, para que o navegador possa usar a próxima fonte na lista se a primeira não estiver disponível.

Tamanho de fonte (font-size)

A escala de tamanho de fonte começa com xx-small como o menor valor e vai até xx-large como o maior valor. xx-small é menor do que x-small, small, medium, large, x-large e xx-large, respectivamente.
Esses valores são relativos e podem variar dependendo do navegador e do dispositivo do usuário. No entanto, a maioria dos navegadores mantém uma relação consistente entre eles, permitindo que você defina um tamanho de fonte relativo com base em uma escala predefinida.

Você pode especificar o tamanho da fonte de várias maneiras:

  1. Tamanhos Absolutos:

    • Pixels (px): Define o tamanho do texto em pixels.
    • Picas (pc): Uma pica é equivalente a 12 pontos.
    • Pontos (pt): Define o tamanho do texto em pontos (1 ponto é aproximadamente igual a 1/72 de polegada).

  2. Tamanhos Relativos:

    • Porcentagem (%): Define o tamanho da fonte em relação ao tamanho do texto do elemento pai.
    • Em (em): Define o tamanho da fonte em relação ao tamanho do texto do elemento pai.
    • Rem (rem): Define o tamanho da fonte em relação ao tamanho do texto do elemento raiz (normalmente o elemento ).

  3. Tamanhos Relativos à Visualização:

    • vw: Define o tamanho da fonte em relação à largura da janela de visualização.
    • vh: Define o tamanho da fonte em relação à altura da janela de visualização.

  4. Tamanhos Relativos ao Texto:

    • larger: Aumenta o tamanho da fonte em relação ao tamanho do texto do elemento pai.
    • smaller: Diminui o tamanho da fonte em relação ao tamanho do texto do elemento pai.

line-heigth

é usada para definir a altura da linha em um elemento de texto. Ela controla o espaço entre as linhas de texto dentro do elemento.
Existem várias maneiras de definir o valor para line-height:

Valor normal: Isso define a altura da linha como normal, o que geralmente é cerca de 1.2 a 1.4 vezes a altura da fonte.

Valor específico: Você pode definir um valor específico em pixels, emems, ou como um múltiplo da altura da fonte.

Um line-height maior resultará em mais espaço entre as linhas de texto, enquanto um line-height menor resultará em menos espaço entre as linhas. O valor que você escolhe depende do estilo de design que você está buscando e da legibilidade do texto em seu contexto.

Alinhamento de Texto (text-aling)

Essa propriedade é responsavel por alinhar nosso texto no nosso elemento.

  • text-align: left;

    left: Alinha o texto à esquerda.

  • text-align: right;

    right: Alinha o texto à direita.

  • text-align: center;

    center: Alinha o texto no centro.

  • text-align: justify;

    justify: Distribui o texto uniformemente entre as margens esquerda e direita, criando espaços adicionais entre as palavras, exceto a última linha.

  • text-align: initial;

    initial: Define a propriedade para o seu valor inicial.

  • text-align: inherit;

    inherit: Herda o valor da propriedade de seu pai.

Decoração de texto (text-decoration)

Adiciona ou remove linhas do nosso texto

  • text-decoration: none

    Remove qualquer decoração de texto.

  • text-decoration: underline

    Adiciona um sublinhado ao texto.

  • text-decoration: overline

    Adiciona uma linha sobre o texto.

  • text-decoration: line-through

    Adiciona uma linha através do texto.

  • text-decoration: underline overline

    Adiciona um sublinhado e uma linha sobre o texto.

  • text-decoration: underline line-through

    Adiciona um sublinhado e uma linha através do texto.

  • text-decoration: blink

    Faz com que o texto pisque ou alterne entre aparecer e desaparecer. Este valor é menos comum e pode não ser suportado em todos os navegadores.

É uma propriedade abreviada de:

  • text-decoration-line

        text-decoration-line: overline;
        text-decoration-line: line-through;
        text-decoration-line: underline overline;

  • text-decoration-style

    É usada para definir o estilo da linha de decoração de texto.

    • text-decoration-style: solid;

      Define uma linha de decoração sólida.

    • text-decoration-style: double;

      Define uma linha de decoração dupla.

    • text-decoration-style: dotted;

      Define uma linha de decoração pontilhada.

    • text-decoration-style: dashed;

      Define uma linha de decoração tracejada

    • text-decoration-style: wavy;

      Define uma linha de decoração ondulada.

  • text-decoration-color

    É usada para definir a cor da linha de decoração de texto.
