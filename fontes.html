Familia de Fontes

Serifadas (Serif)

  1. Roman: Uma das fontes serifadas mais conhecidas e amplamente utilizadas. Ela é clássica e legível, sendo frequentemente usada em documentos impressos e online.
  2. Georgia: Criada especificamente para ser legível na tela, a Georgia é amplamente utilizada na web para textos longos. Ela possui uma aparência elegante e clássica.
  3. Garamond: Uma fonte clássica com uma longa história, Garamond é conhecida por sua elegância e legibilidade. Ela é frequentemente usada em impressões de alta qualidade, como livros e revistas.
  4. Baskerville: Com traços refinados e uma aparência sofisticada, Baskerville é uma escolha popular para design de impressos e web.
  5. Cambria: Desenvolvida pela Microsoft, Cambria é uma fonte serifada moderna projetada para ser legível em telas de computador. Ela é frequentemente usada em documentos do Microsoft Office.
  6. Palatino: Criada pelo designer Hermann Zapf, Palatino é uma fonte clássica com uma aparência elegante e legibilidade sólida. Ela é usada em uma variedade de contextos, desde textos impressos até designs digi

Font-family

Em CSS, a propriedade font-family é usada para especificar a família de fontes que será aplicada a um elemento HTML. Uma família de fontes consiste em um conjunto de fontes semelhantes que compartilham um design básico, mas podem variar em estilo, peso e largura.


Quando você define a propriedade font-family, você está instruindo o navegador a procurar por uma fonte específica no sistema do usuário para exibir o texto. Se a fonte especificada não estiver disponível no sistema, o navegador tentará encontrar uma alternativa similar na lista de fontes fornecida.


A propriedade font-family pode conter vários valores de fonte, separados por vírgulas. Isso permite que você forneça uma lista de fontes de fallback, para que o navegador possa usar a próxima fonte na lista se a primeira não estiver disponível.


Por exemplo, aqui está como você pode usar a propriedade font-family para definir a família de fontes em um elemento HTML:


Exemplo:

.elemento {
font-family: Arial, sans-serif;
}

Font-face (@font-face)

A propriedade font-face é uma regra CSS que permite aos desenvolvedores de sites especificarem uma fonte personalizada para ser baixada e usada em seus sites. Isso é útil quando a fonte desejada não está disponível nos sistemas dos usuários ou quando você deseja usar uma fonte específica que não é comum.


A sintaxe básica da regra @font-face é a seguinte:


Exemplo:01

Passo-1

@font-face {
font-family: 'Nome da Fonte'; /* Define o nome da fonte */
src: url('caminho/para/a/fonte.woff2') format('woff2'), /* Especifica o caminho da fonte e o formato */
url('caminho/para/a/fonte.woff') format('woff'); /* Adiciona alternativas de formato */
}
Passo-2
/* Aplica a fonte personalizada a elementos com a classe "texto-personalizado" */
.texto-personalizado {
font-family: 'Nome da fonte', sans-serif;
}

Para usar arquivos de fonte específicos para diferentes tipos (como negrito, itálico, etc.), você pode definir múltiplas regras @font-face para cada variação da fonte. Aqui está um exemplo de como fazer isso:

src: url('caminho/para/a/fonte-regular.woff2') format('woff2'),
url('caminho/para/a/fonte-regular.woff') format('woff');
font-weight: normal; /* Define o peso da fonte como normal */
font-style: normal; /* Define o estilo da fonte como normal */
}

/* Aplica a fonte personalizada a elementos com a classe "texto-personalizado" */

font-family: 'MinhaFonte', sans-serif; /* Aplica a fonte regular */ font-weight: normal; /* Define o peso da fonte como normal (opcional, já definido na regra @font-face) */

Tamanho de fonte (font-size)

A escala de tamanho de fonte começa com xx-small como o menor valor e vai até xx-large como o maior valor. xx-small é menor do que x-small, small, medium, large, x-large e xx-large, respectivamente.
Esses valores são relativos e podem variar dependendo do navegador e do dispositivo do usuário. No entanto, a maioria dos navegadores mantém uma relação consistente entre eles, permitindo que você defina um tamanho de fonte relativo com base em uma escala predefinida.

font-size: xx-small;

xx-small

font-size: x-small;

x-small

font-size: small;

small

font-size: medium;

medium

font-size: large;

large

font-size: x-large;

x-large

font-size: xx-large

xx-large

Você pode especificar o tamanho da fonte de várias maneiras:

  1. Tamanhos Absolutos:

    • Pixels (px): Define o tamanho do texto em pixels.
    • Picas (pc): Uma pica é equivalente a 12 pontos.
    • Pontos (pt): Define o tamanho do texto em pontos (1 ponto é aproximadamente igual a 1/72 de polegada).

  2. Tamanhos Relativos:

    • Porcentagem (%): Define o tamanho da fonte em relação ao tamanho do texto do elemento pai.
    • Em (em): Define o tamanho da fonte em relação ao tamanho do texto do elemento pai.
    • Rem (rem): Define o tamanho da fonte em relação ao tamanho do texto do elemento raiz (normalmente o elemento ).

  3. Tamanhos Relativos à Visualização:

    • vw: Define o tamanho da fonte em relação à largura da janela de visualização.
    • vh: Define o tamanho da fonte em relação à altura da janela de visualização.

  4. Tamanhos Relativos ao Texto:

    • larger: Aumenta o tamanho da fonte em relação ao tamanho do texto do elemento pai.
    • smaller: Diminui o tamanho da fonte em relação ao tamanho do texto do elemento pai.
